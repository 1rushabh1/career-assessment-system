# Career Assessment & Recommendation System

A MERN stack based psychometric career guidance platform.

## Features

- User Authentication
- Student Profile
- Psychometric Assessment
- Career Recommendation Engine
- Stream Recommendation
- Skill Recommendation
- Course Path Guidance
- Career Dashboard

## Tech Stack

Frontend:
React

Backend:
Node.js
Express.js

Database:
MongoDB
